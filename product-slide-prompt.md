# プロダクト紹介スライドプロンプト
## わくわく探検ワールド（Ocean Adventure）

---

## 📋 Presentation Brief

**Title**: わくわく探検ワールド - 遊びながら学べる、5つの世界の冒険アプリ

**Topic / Scope**:
子どもたちが楽しく遊びながら環境問題や自然科学を学べる、インタラクティブ3D教育アプリケーション。海・自然・空・宇宙・花火の5つの世界で、触って、観察して、体験しながら学ぶことができます。

**Key Messages**:
1. 「環境教育がむずかしい」「子どもが興味を持たない」という課題
2. 遊びながら自然に学べる、体験型の教育アプローチ
3. 5つの世界で多様な学習体験（海洋汚染、生態系、天文学、創造性）
4. 直感操作で3歳から遊べる、家庭でも教育現場でも活用可能
5. 無料で今すぐ始められる、完全ブラウザベースのアプリ

**Style Guide**:
- トーン：親しみやすい、わくわく感、教育的信頼感
- カラーパレット：
  - 海：シアン〜ブルー（#4FACFE → #00A8E8）
  - 自然：グリーン〜エメラルド（#4ADE80 → #10B981）
  - 空：スカイブルー〜パープル（#7DD3FC → #A855F7）
  - 宇宙：パープル〜ブラック（#7C3AED → #000000）
  - 花火：インディゴ〜パープル（#4F46E5 → #9333EA）
- 画像スタイル：実際のアプリ画面、子どもの操作動画、Before/After比較、かわいいマスコットキャラクター

- 「お子さん」「保護者の方」と親しみを込めた呼びかけ

### デモのポイント
